# /assets

Place the following files here before launch:

- `favicon.ico` — browser tab icon (referenced in index.html)
- `favicon.png` — 32×32 PNG fallback
- `og-image.jpg` — Open Graph image for social sharing (1200×630px recommended)
- `logo.svg` — vector logo (optional, for future use)
- `logo.png` — PNG logo fallback (512×512 minimum)
